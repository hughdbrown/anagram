    "IM", "E", "SCN", "EGI", "CH", "E", "S", "T", "F", "T"

In this case, you pick one word from each column to produce a mnemonic. I thought that 'SEMEN THEFT' was a strong contender for a mnemonic but my wife asked me to try something different.

The third program, mnemonic_builder(3).py, requires that the first word be 'THE' and the second word can be any seven-letter word that uses the remaining letters from the mnemonic candidate. In this case, there were only two words that fit:

    ['IENIHESTFT'] EEFHIINSTT EFIINST THE ['FINITES', 'NIFTIES']
    
The wife was happy to go with 'THE NIFTIES'.

The dictionary I used was a 1.7 MB file I found here:

    http://addagram.mytestbench.com/WORD.LST

All the programs will accept any other word list in a text file with one word per line.
